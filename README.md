## Local Usage

If you want to run this locally you'll need `nodejs`/`npm` installed. Clone the repo, then;

```
git clone https://github.com/TiagoaReis/app_finedrinks.git
npm install
yarn -i
```

For run

```
npx react-native run-android
```
